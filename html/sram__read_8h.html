Detailed Description

Low-level driver to read from an HM62256LP-70 SRAM.

Contains the implementation of data_read() which reads a single byte from an external SRAM using the pin configuration specified in config/pin_bus.c.

This module is part of the SRAM Driver group, together with sram_write.c, sram_gpio.c
